Class: Fridge

Fridge()

Simple selecting serialisation/deserialisation of a JS object graph to stand-alone JSON. Does not handle function references. Full restoration of objects requires class prototypes to be passed in to thaw(). The object property prefix 'IB' is reserved for use by this module, and note also that fields who's names start with _ will not be serialised.

Note that the objects being frozen are 'interfered with' by the addition of an _IB_ID field that indicates their 'frozen ID'. This is a (clumsy) solution to the lack of ES7 value objects in ES6. The frozen version (JSONified) version of objects are decorated with fields as follows: _IB_CN: constructor name _IB_REF: the _IB_ID of another object being referenced Date objects are serialised to string.
